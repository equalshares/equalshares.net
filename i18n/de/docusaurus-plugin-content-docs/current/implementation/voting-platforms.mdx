# Online-Plattformen zur Stimmabgabe

Im Folgenden werden einige gängige Softwarepakete für den Betrieb von Online-Plattformen zur Stimmabgabe für Bürgerbudgets vorgestellt und ihre Kompatibilität mit der Methode der gleichen Anteile erläutert.

:::info

Jede Plattform, die eines der vielen mit der Methode der gleichen Anteile kompatiblen [Stimmzettel-Formate](input-formats) nutzt, kann verwendet werden. Nach der Abstimmungsphase müssen die Stimmen einfach aus der Plattform exportiert werden, und das Abstimmungsergebnis kann dann mit einem der verfügbaren [Open-Source-Softwarepakete](computation) für die Methode der gleichen Anteile berechnet werden.

:::

Sie können uns für weitere Ratschläge auch gerne [kontaktieren](../contacts).

## Stanford Participatory Budgeting Platform

Die [Stanford Participatory Budgeting Platform](https://pbstanford.edu/) wird von dem [Stanford Crowdsourced Democracy Team](https://voxpopuli.stanford.edu/) an der Stanford University entwickelt.
Das Projekt ist Open-Source und kostenfrei.
Es wird von diversen Städten in Nordamerika und anderswo benutzt.

Städte, die die Stanford-Plattform nutzen, können diese problemlos mit der Methode der gleichen Anteile kombinieren, indem sie die abgegebenen Stimmen exportieren und dann die Wahlergebnisse mit einem separaten Open-Source-Softwarepaket berechnen.

Es ist einfach, die Wahldaten aus der Verwaltungsoberfläche der Stanford PB-Plattform zu exportieren, unter
* Analytics > Export to Excel > Individual votes. 

Dadurch wird eine `.csv`-Datei erzeugt, die leicht von anderer Software verwendet werden kann.

## Decidim

[Decidim](https://decidim.org/) ist eine beliebte Plattform, die Bürgerbeteiligung in verschiedenen Formen ermöglicht, darunter auch Bürgerbudgets.
Sie wurde zunächst von Madrid und dann von Barcelona verwendet, und danach in vielen anderen europäischen Städten.
Für Bürgerbudgets unterstützt es Abstimmungen bei denen die Wähler für so viele Projekte stimmen können, wie sie möchten, solange sie in das Budget passen. Alternativ kann Decidim eine maximale Anzahl von Stimmen benutzen.

Leider gibt es in Decidim keine einfache Möglichkeit, die abgegebenen Stimmen zu exportieren, so dass ein Systemadministrator die Decidim-Datenbank direkt abfragen muss, um die Daten zu erhalten, die zur Berechnung des Wahlergebnisses bei der Methode der gleichen Anteile erforderlich sind.
